## 🚀 Digital Initiation

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/neural-matrix-portfolio.git
   cd neural-matrix-portfolio
   ```

2. **Initialize dependencies**

   ```bash
   npm install
   ```

3. **Activate development mode**

   ```bash
   npm run dev
   ```

4. **Compile for production**
   ```bash
   npm run build
   ```
